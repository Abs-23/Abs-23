## 🚀 A Couple of Projects

### 🚕 NYC Taxi Trip Dashboard
A personal project analyzing NYC taxi trip data to practice building reporting pipelines and dashboards, and to get hands-on with a newer data platform beyond my usual toolset.

- Cleaned and organized raw trip data into a structured format for reporting.
- Built Power BI dashboards covering trip volume, revenue trends, and demand patterns.

### 💳 Customer Data Project
A learning project simulating how a business might bring together customer data from a few different sources into one consistent view.

- Organized and deduplicated customer records from multiple simulated sources.
- Built simple customer segments (e.g., high value, at risk) to practice turning raw data into something a business team could act on.
- Practiced documenting the process so someone else could follow the same steps.
